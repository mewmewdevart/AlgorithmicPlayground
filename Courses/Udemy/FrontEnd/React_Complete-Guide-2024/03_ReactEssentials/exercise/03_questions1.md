Claro, aqui estão as perguntas e respostas em português:

# Pergunta 1:
## Pergunta:
Da perspectiva de um desenvolvedor: Qual é uma vantagem de construir uma interface de usuário combinando componentes?
## Resposta:
- **Vantagem:** Pequenos blocos de construção reutilizáveis

# Pergunta 2:
## Pergunta:
Ao trabalhar com React: Como um novo componente é definido?
## Resposta:
- **Definição:** Criando uma função que retorna código JSX

# Pergunta 3:
## Pergunta:
Qual é o propósito do JSX?
## Resposta:
- **Propósito:** Permite que você defina o código HTML alvo dentro de seus componentes

# Pergunta 4:
## Pergunta:
Qual afirmação sobre JSX é VERDADEIRA?
## Resposta:
- **Afirmação Verdadeira:** O código JSX é usado para definir o código UI/HTML alvo

# Pergunta 5:
## Pergunta:
O que o React faz com os componentes que você usa no código JSX?
## Resposta:
- **Ação do React:** Deriva uma árvore de componentes que é então usada para executar comandos que atualizam o DOM do site

# Pergunta 6:
## Pergunta:
Como você costuma usar componentes personalizados?
## Resposta:
- **Uso Típico:** Você usa componentes personalizados como elementos HTML dentro do código JSX

# Pergunta 7:
## Pergunta:
Qual afirmação sobre componentes personalizados é FALSA?
## Resposta:
- **Afirmação Falsa:** Componentes personalizados não precisam ser definidos ao lado dos componentes que desejam usá-los
